# AST SYSTEM - 硬核越野数据中心小程序

## 项目简介
AST SYSTEM 是一款专为越野跑爱好者打造的“硬核越野数据中心”微信小程序。该项目不仅提供丰富的越野赛事资讯与路线详情，还具备低功耗蓝牙（BLE）设备连接功能，能够与用户的智能运动腕表进行互联与数据交互。整个项目采用了沉浸式的全局暗黑主题设计（Dark Mode），完美契合户外极客与越野跑者的硬核风格。

## 主要功能特性

### 1. 蓝牙设备互联 (BLE Connect)
* **设备扫描与连接**：支持搜索附近的蓝牙腕表设备，并附带动态的雷达波纹扫描动画视觉效果。
* **状态同步**：连接成功后，可在首页实时查看已绑定的设备名称（如“AST的RMB PRO”），以及“已连接”、“100%电量”和“正常”等设备运行健康状态。

### 2. 赛事中心 (Race Center)
* **赛事浏览**：首页与列表页展示了即将举办的超级越野赛（如“广州莫六公超级越野赛”、“东莞100超级越野赛”等），包含赛事名称、时间、地点及 ITRA 认证角标。
* **赛事详情解析**：
  * **多组别切换**：支持同一赛事下不同公里数（如 12km, 21km, 32km, 48km）的无缝切换浏览。
  * **关键数据展示**：直观展示所选组别的发枪时间、实际距离、爬升下降数据及关门时间。
  * **路线地图与计划**：提供高清晰度赛道地图和海拔图展示，并预留了“制定计划”功能模块。

### 3. 用户中心与鉴权 (User Profile & Auth)
* **账号鉴权系统**：包含完整的登录与注册界面，支持手机号/邮箱及密码的身份验证。
* **个人看板**：未登录状态下，系统会引导用户登录以同步比赛与计划数据；登录后展示用户的个性化信息（如默认昵称“AST 探险家_007”）。
* **快捷导航**：提供“我的设备”、“历史数据”及“系统设置”的快捷入口。

### 4. UI/UX 亮点
* **硬核暗黑视觉**：全局以纯黑色（`#000000`）为背景底色，搭配白色文本及品牌专属橙色（`#FF9811`）高亮，视觉冲击力强。
* **完全自定义界面**：摒弃了微信原生顶部导航栏和底部 TabBar，采用了沉浸式的自定义导航栏和悬浮式底部双 Tab（“我的产品”、“个人中心”），提供了高度统一的品牌交互体验。
